Introducing... Jargon

Currently, I’m working on an experimental programming language of my own as a side project primarily for the purpose of learning about compilers and systems in depth. I have an online compiler and REPL for it on here so if you’re curious, check it out. 

import "std/console";


import "std/math";




struct Vector2: type = {                    


field x: real;


field y: real;


};




proc Vector2::magnitude(): real() = math::sqrt(self::x ** 2 + self::x ** 2);




proc Vector2::normalized(): real() = {


self::x /=  self::magnitude();


self::y /=  self::magnitude();


};




proc main(param args: string[_]): int() =  {


var position: Vector2 = { 10.5, 43.2 };


console::print(position::magnitude());


console::print(position::normalized());


return 0;


};
